```jsx
<SearchLookUpField
  valueMember="id"
  displayMember="partCode"
  rows={rows}
  columns={columns}
  value={value}
  onChangeValue={handleSearchFieldChange}
/>
```
- ValueMember is the actual value from the field
- DisplayMember is the display value on the field
